Welcome to hulse-py’s documentation!¶

hulse-py lets you run queries on pre-trained NLP models using Hulse.

NLP models from the Hugging Face Transformers library are currently supported. Please refer to the Hugging Face’s documentation for more in-depth information about the models and tasks covered.

Hulse is currently in beta.
